# Domain Modules

This directory will hold feature modules in production order:

- `auth` (existing; to be hardened with session/device management)
- `access-control`
- `organization`
- `employees`
- `attendance`
- `leave`
- `payroll`
- `notifications`
- `audit`
- `reports`
- `settings`

Shared platform capabilities belong in `src/platform` and `src/common`.
